# App README
## Getting Started

1. Clone the repository:
   git clone https://github.com/akshatg1349/EcoSortAIApp.git
   cd EcoSortAIApp
2. In ChatService.swift, please put your OpenAI API key in 'apiKey' variable. This is required for AI Chat Assistant functionality to work.
3. Build and run the app: Open EcoSortAIApp.xcodeproj (or the .xcworkspace if present) in Xcode, select the target device or simulator, then press Cmd+R to build and run.
